Detailed Description

Implement to define a network protocol. Protocols define what data to transmit, and how and where it can be transmitted.

Shorthand to send a request for a default instance of this protocol's data from a client. Requires SetClientDefaults() to be implemented.

Parameters
toWhoMain.player index of player (client) being requested for this data. -1 for all clients.
ignoreWhoMain.player index of player (client) being ignored. -1 for no client.
