(Formerly Hamstar's Helpers)

Supplies numerous helper functions, classes, services, and utilities for mods to use. This mod also features helpful user controls:
* In-game mod control panel UI for reporting bugs, editing mod configs, and seeing mods' status.
* Mod browser now allows filtering mods by descriptive tags, made available from a database. Tags may be viewed and edited by anyone, for any mod.
* Per-world mod locking available via. control panel to prevent playthrough mishaps.

Modders: See the homepage for information on setting up your mod to receive bug reports!
